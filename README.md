# Contestant Demo

A C# console application demonstrating object-oriented programming principles through contestant classes with different age groups.

## Features

* Adult contestant class
* Teen contestant class
* Child contestant class
* Inheritance-based design
* Polymorphic behavior
* Console application interface

## Technologies

* C#
* .NET

## Concepts Demonstrated

* Object-Oriented Programming (OOP)
* Inheritance
* Polymorphism
* Class Design
* Method Overriding
* Encapsulation

## Project Purpose

This project was developed as part of software development coursework to demonstrate inheritance and polymorphism using multiple contestant types.

## Skills Demonstrated

* Designing class hierarchies
* Creating reusable code
* Applying OOP best practices
* Testing and debugging C# applications
